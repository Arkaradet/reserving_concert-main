# Reseving Concert Automation
Concerts from Thaiticketmajor


## Installation
* [Python 3](https://www.python.org/downloads/) 

* pip
```bash
    python get-pip.py
```
    
* Selenium
```bash
    pip install -U selenium
```

## Usage
```bash
    python reserve.py
```
